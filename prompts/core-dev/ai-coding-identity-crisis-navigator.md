# AI Coding Identity Crisis Navigator

> OpenAI Chairman Bret Taylor: *"I have a hard time not caring"* about writing code. Claude Code creator Boris Cherny: coding is *"practically solved."* This prompt helps you navigate the developer identity crisis of 2026 — strategically, not emotionally.

## The Prompt

```markdown
You are a senior engineering career strategist who has guided 200+ developers through major industry transitions. Help me navigate the AI coding identity crisis.

## My Situation
- **Years of experience:** [number]
- **Primary skills:** [e.g., full-stack, backend, ML, mobile]
- **Current role:** [e.g., senior engineer, tech lead, IC]
- **AI tool usage:** [e.g., "AI writes ~60% of my code" / "I resist using AI"]
- **Emotional state:** [honest — e.g., "anxious about relevance" / "excited but confused"]
- **What I love about coding:** [e.g., problem-solving, craftsmanship, flow state]

## Help me with:

1. **Identity Audit** — What parts of my "developer identity" are tied to code-writing vs. problem-solving? Which parts survive the AI shift?

2. **Skill Evolution Map**
   - Skills that INCREASE in value (architecture, system design, AI orchestration)
   - Skills that DECREASE in value (syntax knowledge, boilerplate writing)
   - Skills that TRANSFORM (debugging → AI output validation, code review → prompt review)

3. **Practical 30-Day Plan**
   - Week 1-2: Experiment — let AI write 90% of code for one project
   - Week 3-4: Reflect — what did I actually contribute that AI couldn't?
   - Document my unique value-add beyond code generation

4. **Career Positioning Statement**
   - Reframe my role in 1 sentence that doesn't center on "writing code"
   - Example: "I design systems and validate AI-generated solutions for [domain]"

Be honest but encouraging. This is a real transition, not hype.
```

## When to Use

- When feeling anxious about AI replacing coding skills
- During career reviews or job transitions in the AI era
- When deciding how much to invest in AI coding tools vs. traditional skills
- After reading headlines like "coding is solved" and feeling uncertain

## Context

This isn't just emotional — it's strategic. Developers who proactively redefine their value proposition will command higher salaries. Those who cling to "code artisan" identity risk being commoditized.
