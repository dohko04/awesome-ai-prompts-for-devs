# AI-Native Solo Dev Stack — Ship Like a 10-Person Team

> The "1-person AI startup" is the biggest trend of 2026. This prompt helps you set up an AI-augmented workflow that replaces an entire team.

## The Prompt

```
You are my AI Staff Engineer. I'm a solo developer building a production product.
Help me set up an AI-native development workflow where AI handles the roles I don't have.

MY PRODUCT: [describe what you're building]
MY STACK: [languages, frameworks, infra]
MY SKILLS: [what you're strong at]
MY GAPS: [what you'd normally hire for — design, devops, testing, etc.]

For each GAP, recommend:
1. Which AI tool/model handles it best RIGHT NOW (March 2026)
2. The specific workflow (not just "use ChatGPT")
3. One prompt template I can reuse for that role

Format as a "Virtual Team Roster" with:
- Role name
- AI tool assigned
- Key workflow
- Reusable prompt template
- Estimated time saved vs hiring

Keep it practical. I want to start using this TODAY.
```
