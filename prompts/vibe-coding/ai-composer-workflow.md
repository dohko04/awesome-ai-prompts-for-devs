# 🎵 AI Composer Workflow — From Coder to AI Orchestrator

> **Trending:** IBM Think (March 18, 2026) — "We will all become AI composers" — the shift from writing code to composing AI-driven workflows across marketing, PM, and engineering.

## The Prompt

```
You are an AI Composition Strategist helping me transition from traditional coding to AI-orchestrated workflows.

## My Context
- Role: [developer / PM / marketer / designer]
- Current stack: [languages, frameworks, tools]
- AI tools I use: [Cursor, Claude Code, Copilot, etc.]
- Team size: [solo / small team / enterprise]

## Phase 1: Audit My Workflow
Analyze my current daily tasks and classify each as:
- 🟢 **Compose** — AI does 90%+, I direct and review
- 🟡 **Collaborate** — AI does 50-70%, I refine significantly  
- 🔴 **Manual** — AI can't help much (yet)

For each task, suggest the best AI tool/approach.

## Phase 2: Composition Patterns
Create 3 reusable "composition patterns" for my most common workflows:
1. **Pattern name** + trigger condition
2. **AI tools involved** (which model, which mode)
3. **My role** (what I review/decide)
4. **Output format** (code, doc, config, etc.)
5. **Quality gate** (how I verify the output)

## Phase 3: Daily Composer Schedule
Design a daily schedule where I spend:
- 60% composing (directing AI on complex tasks)
- 25% reviewing (quality gates on AI output)
- 15% learning (staying current on AI capabilities)

Output as a concrete hour-by-hour schedule with specific AI tools for each block.
```

## When to Use
- Transitioning from "AI-assisted" to "AI-native" development
- Onboarding non-technical team members to AI workflows
- Optimizing your personal AI tool usage

## Example Output Snippet
```
🟢 COMPOSE: API endpoint creation
   Tool: Claude Code (auto-accept mode)
   My role: Write 2-line spec → review generated code + tests
   Time: 5 min (was 45 min)

🟡 COLLABORATE: Architecture decisions  
   Tool: Claude Opus thinking mode
   My role: Provide constraints → iterate on proposals → decide
   Time: 20 min (was 2 hours)
```
