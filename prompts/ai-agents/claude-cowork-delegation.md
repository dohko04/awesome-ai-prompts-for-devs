## The Prompt

```markdown
You are an expert at designing AI agent delegation workflows using Claude Cowork and similar agentic platforms. Help me set up effective task delegation for my team.

## My Context
- **Team type:** [engineering / product / mixed]
- **Current bottleneck:** [e.g., code reviews take 3 days, documentation is always stale, onboarding is slow]
- **Tools we use:** [e.g., GitHub, Linear, Notion, Slack]
- **MCP servers available:** [list any connected tools]

## Design a Delegation Framework:

### 1. Task Classification
Categorize my team's work into:
- 🤖 **Fully delegatable** — AI handles end-to-end, human reviews output
- 🤝 **Co-work** — AI does 80%, human guides and approves
- 👤 **Human-only** — Requires judgment, creativity, or stakeholder trust

### 2. Agent Workflows (pick top 3 bottlenecks)
For each, design:
- Trigger (what starts the workflow)
- Agent steps (what the AI does)
- Checkpoints (where humans review)
- Output (what gets delivered)

### 3. Guardrails
- What should agents NEVER do autonomously?
- Approval gates for sensitive operations
- Escalation paths when agents get stuck

### 4. Measurement
- How to track time saved
- Quality metrics (error rate before/after)
- Team adoption metrics
```

## Sample Use Case

**Before:** Developer writes PR → waits 2 days for review → reviewer spends 45 min → back and forth
**After:** Agent pre-reviews in 2 min → flags real issues → reviewer spends 10 min on what matters
