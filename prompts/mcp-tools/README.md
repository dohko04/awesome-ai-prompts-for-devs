# 🔌 MCP (Model Context Protocol) Prompts

Build MCP servers and tools that extend AI assistants with custom capabilities. MCP is the standard for connecting LLMs to external tools, data sources, and APIs.

| Prompt | Description |
|--------|-------------|

## Why MCP?

MCP is becoming the universal standard for AI tool use in 2026. If you build developer tools, APIs, or internal platforms — exposing them via MCP means any AI assistant (Claude, Cursor, Copilot) can use them natively.
