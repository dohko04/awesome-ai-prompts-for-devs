# Contributing to Awesome AI Prompts for Devs

Thanks for wanting to contribute! 🎉

## How to Add a Prompt

1. Fork this repo
2. Add your prompt to the appropriate category in `README.md`
3. Follow the format:

```markdown
### N. Prompt Title

> **When to use:** Brief description.

\```
Your prompt template here
\```

<details>
<summary>💡 Example Output</summary>
Show a realistic example of what the AI returns.
</details>
```

4. Submit a Pull Request

## Rules

- **Developer-focused only** — no generic prompts
- **Include example output** — show, don't tell
- **Test with 2+ LLMs** — ensure it's model-agnostic
- **Keep it practical** — prompts should solve real problems

## Questions?

Open an issue. We're friendly. 🐉
