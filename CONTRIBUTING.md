# 🤝 Contributing to Awesome AI Prompts for Devs

Thanks for wanting to contribute! 🎉

## How to Add a Prompt

1. **Fork** this repository
2. **Choose the right category** folder inside `prompts/` (or propose a new one)
3. **Create a new `.md` file** with a kebab-case name (e.g., `my-awesome-prompt.md`)
4. **Follow this format:**

```markdown
# 🎯 Prompt Title

> **When to use:** One-line description of when this prompt is useful.

## Prompt Template

\```
Your prompt template here.
Use [brackets] for variables the user should fill in.
\```

## Example Output

Show a realistic example of what the AI returns when using this prompt.

---

> 💡 **Want 100+ more prompts like this?** → [AI Dev Toolkit ($9)](https://ai-dev-toolkit-five.vercel.app)
```

5. **Update the category `README.md`** to include your new prompt in the table
6. **Submit a Pull Request** with a brief description of why this prompt is useful

## Guidelines

- ✅ Prompts must be **developer-focused** (no generic "write me a poem" stuff)
- ✅ Include a **real example output** — show, don't tell
- ✅ Test with at least **2 different LLMs** before submitting
- ✅ Keep prompts **model-agnostic** (no ChatGPT-specific or Claude-specific syntax)
- ✅ One prompt per file
- ❌ Don't modify other people's prompts without discussion

## Proposing a New Category

If your prompt doesn't fit any existing category:

1. Open an issue first to discuss
2. Create the folder with a `README.md` following the existing pattern
3. Add at least 2 prompts to justify a new category

## Questions?

Open an issue. We're friendly. 🐉
